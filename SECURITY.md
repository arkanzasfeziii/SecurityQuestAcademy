# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 2.0.x   | :white_check_mark: |

## Reporting a Vulnerability

1. **Do not** open a public issue.
2. Contact [@arkanzasfeziii](https://github.com/arkanzasfeziii) directly.

**Note:** This is an educational platform. Challenges intentionally teach offensive techniques. Report only vulnerabilities in the platform itself (e.g., sandbox escape, arbitrary code execution outside the intended exec engine).
